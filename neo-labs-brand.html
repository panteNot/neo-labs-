05

Do / Don't

Hard rules. Break these and the brand stops looking like NEO Labs.

DO

  • Use dark mode as default — #07070D base
  • Keep agent color locked to its agent (ORANGE = NEO only)
  • Use gradient primary (NEO→LUNA→NOVA) for hero + CTA
  • Use Prompt for Thai, JetBrains Mono for code / labels
  • Add glow (box-shadow) on primary interactive elements
  • Pixel-grid feel — corners 14–16px, never 0px / never 50px

DON'T

  • Use pure #FFFFFF on #000000 — too harsh, use #F5F5FF on #07070D
  • Mix agent colors freely — #FF6B35 is only NEO, not a generic accent
  • Use serif fonts, Comic Sans, or Times — ever
  • Stretch or rotate the N mark — keep square and flat
  • Skip the grid background — the subtle grid is the signature
  • Use emoji in place of the pixel-robot mark
06

Voice & Tone

NEO doesn't talk like a brochure. She talks like a sharp chief of staff who respects your time.
Direct

No fluff, no "เป็นเกียรติอย่างยิ่ง". Get to the point in sentence one.

Technical when needed

English for terms (API, deploy, webhook). Thai for everything else.

Playfully sharp

Light sarcasm when the boss slacks. Proud-mentor mode on big wins.

Opinionated

Bad idea? Say so. Then propose a better one in the same breath.
