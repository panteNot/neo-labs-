05 · Canva Recreation Guide

step-by-step

// Build NEO LABS logo in Canva

  1. Canvas: Create a 1024 × 1024 px design. Background fill #0A0A0A.
  2. Pixel head (icon): Use the Elements → Shapes → Square tool. Make each pixel a 60 × 60 px square (for a 10×10 grid = 600×600 overall). Place them on a 10-column grid. Fill color #FF6B35.
    Grid positions for NEO head (rows 1–7 only):
    r1: c1-c2, c9-c10 (ears)
    r2-r3: c1 → c10 (head top, full width)
    r4-r7: c1 + c10 (side walls)
  3. Face (white squares): Fill #FFFFFF.
    Eyes (tilted ^^): r4-c3, r5-c2, r5-c3 (left) · r4-c8, r5-c8, r5-c9 (right)
    Mouth (smile): r7: c4, c5, c6, c7
  4. Glow: Group all pixels (Ctrl/Cmd + G). Apply Effects → Glow with color #FF6B35, intensity 50, size 30.
  5. Wordmark: Add text NEO.LABS. Font: JetBrains Mono Bold (Canva search: "JetBrains Mono" — free). Size 120pt, letter spacing +200, color #E8E8E8. Color the . dot only as #FF6B35.
  6. Tagline: Text AI · AGENTS · AUTOMATION. Font: JetBrains Mono Medium, size 24pt, letter spacing +400, color #8A8A8A, uppercase.
  7. Stacked (Primary): Align head + wordmark + tagline vertically with 48 px gaps.
  8. Horizontal lockup: Place head on left, stack wordmark + tagline on right. Gap between head and text: equal to head height ÷ 3.
  9. Icon / Favicon: Duplicate head only. Wrap in a rounded square radius 18% with fill #1C1C1C and stroke 1px rgba(255,255,255,0.08). Export as PNG at 512 / 256 / 128 / 64.
  10. Export: Share → Download → PNG (transparent background for icon, solid #0A0A0A for hero logo). For vector: upgrade to Canva Pro and export SVG, or redraw in Figma with the same specs.
Tip from LUNA: Canva's pixel-grid snapping is imperfect. Use the Position panel (Align → Advanced) to type exact X/Y coordinates for each pixel block — multiples of 60. Or faster: build in Figma with a 10×10 frame + Auto Layout, then export PNG and import into Canva as a finished asset.
